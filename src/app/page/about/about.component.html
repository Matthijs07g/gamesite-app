Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

GameBlogger

Op de site worden er een soort van blogs bijgehouden over verschillende games. Op deze blogs staat informatie over de games, links naar handige paginaâ€™s en reacties van mensen over de blog of over hun ervaring met de game zelf.

Er wordt gebruik gemaakt van 4 entiteiten zoals te zien is in de ERD.
de entiteit gebruiker maakt gebruik van de attributen emailaddress, wachtwoord, geboortedatum, GebruikersNaam. d.m.v. het emailaddress kunnen we zien of een account uniek is en in combinatie met wachtwoord wordt er ingelogd. de geboortedatum is voor als een game/blog beschikbaar is vanaf een bepaalde leeftijd bijvoorbeeld 16 jaar en ouder.
De blog zelf heeft de attributen: titel, afbeelding, beschrijving en createDate. met de titel en afbeelding krijg je een al snel een indruk over de blog. met de beschrijving kan je uitgebreid lezen waar de game van de blog overgaat.
De game maakt gebruik van de attributen: naam, korteBeschrijving en genre. De naam spreekt voor zich en een korte beschrijving is voor een snelle indruk over de game. En genre is er voor als er later nog een optie komt om op games te sorteren

Entiteiten

Voor de entiteit en attribuut gebruik kunt u dat zien in de uitleg hierboven.
de relaties zijn als volgt:
een gebruiker kan vrienden worden met een andere gebruiker, een gebruiker kan een blog beheren/bezoeken en een gebruiker kan ook reacties plaatsen.
een blog gaat over een game en bevat reacties die onderaan de blog te zien zijn.

Gebruiker

  • emailaddress: string
  • wachtwoord: string
  • geboortedatum: dateTime
  • GebruikersNaam: string
  • Admin: boolean

Blog

  • titel: string
  • afbeelding: img
  • beschrijving: string
  • createDate: date

Game

  • naam: string
  • korteBeschrijving: string
  • genre: Array

Reacties

  • tijd: dateTime
  • inhoud: string
  • like: int
